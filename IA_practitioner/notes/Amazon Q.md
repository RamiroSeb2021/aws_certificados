## Qué es

- Asistente basado en IA generativa para acelerar desarrollo de software y aprovechar datos internos empresariales.
- Productos mencionados: Amazon Q Business, Amazon Q Apps y Amazon Q Developer.
- Integrable con QuickSight, Amazon Connect y AWS Supply Chain.
- Importante en fuente: al usar Amazon Q Developer Pro o Amazon Q Business, el contenido no se usa para mejorar modelos subyacentes para otros.

## Amazon Q Business

- Crea entorno de chat interactivo para usuarios finales de la organización.
- Se integra con Amazon Kendra, Amazon S3, Microsoft SharePoint, Salesforce, etc.
- Usa datos de empresa y conocimiento de LLM, o solo datos de empresa.
- Responde preguntas, genera contenido y completa tareas usando datos empresariales.
- Filtra respuestas generativas según permisos de usuario y grupo.

## Conectores y plugins

- Conectores sincronizan datos desde repositorios de contenido.
- Sincronización automática programada para mantener datos actualizados.
- Se pueden crear conectores personalizados con API de AWS Enterprise Q.
- Plugins permiten acciones como crear tickets en Zendesk, incidentes en ServiceNow, bugs en Jira o casos en Salesforce.

## Amazon Q Apps

- Permite crear aplicaciones sin conocimientos de programación.
- Genera apps desde conversaciones con Q Business o requisitos en lenguaje natural.
- Usa datos internos de empresa para respuestas y acciones.
- Integra con S3, Salesforce, Google Drive, Microsoft 365, etc.
- Amazon Q Library permite acceder, compartir y personalizar apps.

## Amazon Q Developer

- Formalmente CodeWhisperer según fuente.
- Ayuda con sugerencias de código en tiempo real, automatización, refactorización, documentación y modernización.
- Integra con IDEs, línea de comandos, consola AWS, documentación, Slack/Teams vía AWS Chatbot, CodeCatalyst y app móvil de consola.
- Incluye escaneo de vulnerabilidades y optimización de código.

## Amazon Q en otros servicios

- QuickSight: visualizaciones, análisis ad-hoc, alertas de anomalías y conocimientos empresariales.
- Amazon Connect: mejora interacciones con clientes mediante respuestas automatizadas más precisas.

## Claves de examen

- Q Business = chat con datos empresariales y permisos.
- Q Apps = apps sin código desde lenguaje natural.
- Q Developer = asistencia para desarrollo, IDE/CLI y seguridad de código.
- Q puede integrarse en QuickSight y Connect.
