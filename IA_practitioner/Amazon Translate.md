## Qué es

- Servicio de traducción automática neuronal.
- Ofrece traducción rápida, de alta calidad, asequible y personalizable.
- Soporta más de 70 idiomas según fuente.
- Se integra con herramientas AWS como Amazon S3 y AWS Lambda.

## Funcionamiento básico

- Entrada: texto en idioma origen.
- Amazon Translate procesa traducción.
- Salida: texto traducido en idioma destino.

## Caso de uso de fuente

- Entrada: “Hola, este curso es impresionante”.
- Salida: “Hi, this course is awesome”.

## Claves de examen

- Translate = traducción automática neuronal.
- Si el caso habla de traducir texto entre idiomas, elegir Amazon Translate.
- Diferenciar de Comprehend: Comprehend entiende/extrae información del texto; Translate traduce.
